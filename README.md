# Class Imbalance and Its Impact on Machine Learning Performance

## 📌 Overview
This project analyzes how class imbalance affects the performance of machine learning models. It demonstrates how imbalanced datasets can bias model predictions and reduce accuracy for minority classes.

## 🛠 Tools Used
- R Programming
- ggplot2
- caret / tidymodels
- dplyr

## 📊 Techniques Used
- Data preprocessing
- Handling class imbalance (SMOTE / undersampling / oversampling)
- Model training (Logistic Regression / Decision Tree)
- Evaluation using confusion matrix, precision, recall, F1-score

## 📈 Key Insight
Models trained on imbalanced datasets tend to favor the majority class, leading to poor recall for minority classes.

## 🚀 How to Run
1. Open RStudio
2. Run scripts in order:
   - data creation.R
   - Performance Testing.R
